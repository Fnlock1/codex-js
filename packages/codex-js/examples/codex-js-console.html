codex-js 功能控制台

静态 HTML，生成可复制的 PowerShell 命令
本地模式：不输出 JSON 流
浏览器不能直接执行本机 Node 命令。这个页面负责把所有功能对接成命令和请求模板，你复制到 PowerShell 运行即可。

快速任务

适合直接让 codex-js 写文件、改代码、做网页。默认使用交互式摘要模式，不带 --json。

多专家

专家团模式会启用技术 Leader。Leader 先规划专家，可动态创建新专家；每个专家有独立记忆；专家遇到问题先问 Leader，Leader 拿不准才问用户；子专家之间禁止通信。

记忆系统

对接 remember、recall_memory、forget_memory、list_memories。这里生成的是让模型调用记忆工具的任务命令。

工具和线程

常用管理命令，不经过模型，直接查看工具、线程、测试状态。

HTTP Bridge

当前 codex-js 只有 CLI / stdio app-server，没有浏览器可直接调用的 HTTP 服务。这里预留给你以后加本地 bridge。

全局设置

配置会存在浏览器 localStorage。API Key 推荐使用环境变量，不建议写进命令历史。

生成结果 
选择一个功能，然后点击“生成命令”。
